HTML - Hypertext Markup Language
Text content - what you see
Markup - what it looks like
Refernces to other docs - Links to other pages

HTML Markup is made from Tags
<name> content </name> Opening and closing tags
Element - all the tags

Tags <b>content </b> #bold <em>content</em> #emphasize inline

HTML Attributes
Name=”value”
Anchor = <a> <a href= “www.reddit.com”> derp </a> derp now points to www.reddit.com
This is a <a href = "www.w3c.org"> link </a> This is a link – link points to www.w3c.org

Image Tags
<img> - images <img src = “uri” alt = “text”> # text will display if image does not load. Also for broken requests and the blind.
Inline
No closing tag
Void tag – tag with no content

WhiteSpace
All lines in HTML run together
<br> = break Void tag
Inline – ends the line
<p> = paragraph Non-void tag
<p> content </p> Block – puts a box around the text
Box can have height and width

span
<span class = “foo”> text </span> Inline
CSS – changing style or how text looks

div
<div class = “bar”> text </bar> Block
